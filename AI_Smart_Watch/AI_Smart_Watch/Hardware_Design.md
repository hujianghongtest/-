# AI 智能助手手表 — 硬件设计方案

## 项目概述

基于 ESP32-S3-Zero 开发板设计的穿戴式 AI 智能助手手表。具备语音对话、屏幕显示、环境感知、情感交互等功能。

### 核心功能
- **AI 语音对话**：通过 WiFi 连接云端 AI（如 OpenAI / 百度文心 / 阿里通义千问），实现自然语言对话
- **情感关怀**：基于时间、环境数据、交互频率，主动问候并给出生活建议
- **屏幕显示**：圆形/方形 TFT 屏幕，显示时间、温度、天气、AI 对话内容
- **环境感知**：温度、湿度、气压实时采集
- **触控/按键交互**：实体按键 + 可选触摸屏

### 3.2 详细连接表

#### SPI 总线 — TFT 显示屏 (GC9A01 / ST7789)

| 显示屏引脚 | ESP32-S3 引脚 | 说明 |
|-----------|--------------|------|
| VCC | 3.3V | 电源 (3.3V) |
| GND | GND | 地 |
| SCL (SCLK) | GPIO12 | SPI 时钟 |
| SDA (MOSI) | GPIO11 | SPI 主机输出 |
| CS | GPIO10 | 片选 |
| DC | GPIO13 | 数据/命令选择 |
| RST | GPIO14 | 复位 |
| BL (背光) | GPIO47 | PWM 背光控制 |

#### I2C 总线 — BME280 传感器 + DS3231 RTC

| I2C 信号 | ESP32-S3 引脚 | 说明 |
|----------|--------------|------|
| SDA | GPIO17 | I2C 数据线 |
| SCL | GPIO18 | I2C 时钟线 |

I2C 设备地址：
- BME280: 0x76 (SDO 接 GND)
- DS3231: 0x68

> **注意**：I2C 总线需外接 4.7kΩ~10kΩ 上拉电阻至 3.3V。部分模块已自带。

#### I2S 输入 — INMP441 麦克风

| INMP441 引脚 | ESP32-S3 引脚 | 说明 |
|-------------|--------------|------|
| VDD | 3.3V | 电源 |
| GND | GND | 地 |
| SCK | GPIO15 | I2S 位时钟 |
| WS | GPIO16 | I2S 左右通道选择 |
| SD | GPIO7 | I2S 串行数据输出 |
| L/R | GND | 左通道 (接地=左) |

#### I2S 输出 — MAX98357 音频功放

| MAX98357 引脚 | ESP32-S3 引脚 | 说明 |
|--------------|--------------|------|
| VIN | 5V (USB或电池升压) | 功放电源 (2.5V~5.5V) |
| GND | GND | 地 |
| BCLK | GPIO8 | I2S 位时钟 |
| LRC | GPIO9 | I2S 左右通道 |
| DIN | GPIO6 | I2S 串行数据输入 |
| SD | GPIO4 | 关断控制 (低有效，拉高使能) |
| GAIN | GND / 不接 | 增益选择 (详见手册) |

#### 按键输入

| 按键 | ESP32-S3 引脚 | 功能 |
|------|--------------|------|
| BTN_UP | GPIO1 | 上翻 / 音量+ |
| BTN_DOWN | GPIO3 | 下翻 / 音量- |
| BTN_OK | GPIO5 | 确认 / 唤醒 / 语音输入 |

> 按键需外接 10kΩ 下拉电阻，按下接 3.3V（或使用内部上拉，按下接地）。

#### 电池电量检测

| 信号 | ESP32-S3 引脚 | 说明 |
|------|--------------|------|
| BAT_ADC | GPIO2 (ADC1_CH1) | 电池电压经分压后接入 |

> 分压比：锂电池满电 4.2V → 分压后 < 3.3V (推荐 1/2 分压：100kΩ + 100kΩ)

## 四、子系统详细设计

### 4.1 电源管理

```
[USB 5V] → [TP4056] → [LiPo 3.7V 500mAh]
                 │
                 ├→ DW01 保护板 (TP4056 模块自带)
                 │
                 └→ [3.3V LDO] → ESP32-S3 + 外设 (≤500mA)
                        │
                        └→ 100μF 钽电容滤波
```

- **充电**：TP4056 通过 USB Type-C 充电，最大 1A
- **放电保护**：TP4056 模块自带 DW01 锂电保护芯片
- **续航估算**：500mAh / ~80mA (平均功耗) ≈ 6小时
- **待机优化**：ESP32-S3 深度睡眠模式 ~5μA，配合 RTC 唤醒

### 4.2 音频子系统

```
[INMP441 Mic] → I2S → [ESP32-S3] → I2S → [MAX98357] → [Speaker]
                      │
                      ├→ 音频采集: 16kHz, 16bit, 单声道
                      ├→ 回声消除: 软件 AEC (ESP-ADF)
                      ├→ 语音识别: 云端 ASR (HTTP/WebSocket)
                      ├→ AI 回复: 云端 LLM API
                      └→ TTS 合成: 云端 TTS 或本地 ESP-TTS
```

- **麦克风**：INMP441 数字 MEMS，高灵敏度 (-26dBFS)
- **功放**：MAX98357，I2S 输入，3W @ 4Ω
- **语音链路**：本地采集 → 云端 ASR → LLM 推理 → 云端 TTS → 本地播放

### 4.3 显示子系统

```cpp
// 显示初始化参数 (GC9A01 / ST7789)
SPI Freq:     40 MHz (标准模式)
Resolution:   240 × 240
Color Depth:  16-bit (RGB565)
Orientation:  根据佩戴方向软件可调
```

- **UI 框架**：LVGL (Light and Versatile Graphics Library)
- **界面元素**：表盘时钟、温度/天气卡片、AI 对话气泡、设置菜单
- **背光**：PWM 控制，亮度 0~100%，配合环境光自动调节（可选）

## 七、注意事项与建议

### 7.1 电源注意事项
- MAX98357 峰值电流可达 1A，需在 VIN 端并联 100μF 电容
- ESP32-S3 WiFi 发射时功耗约 300-500mA，需确保 3.3V LDO 输出能力 ≥ 500mA
- 电池端建议增加 10μF + 0.1μF 去耦电容

### 7.2 信号完整性
- I2S 时钟线 (SCK/BCLK) 尽量短，远离 WiFi 天线区域
- I2C 总线长度 < 15cm，否则降低速率
- SPI 时钟 40MHz 时，连线 < 10cm

### 7.3 天线注意事项
- ESP32-S3-Zero 板载陶瓷天线
- 手表外壳需为非金属材质（塑料/树脂/3D打印 PLA）
- 金属表带可能影响 WiFi 信号，建议皮/硅胶/尼龙表带

### 7.4 散热
- 持续 WiFi + 音频播放时，ESP32-S3 温度可达 50°C+
- 手表底部与皮肤接触处需做隔热处理
- 建议在外壳上开散热孔

### 7.5 软件架构建议
```
firmware/
├── src/
│   ├── main.cpp           # 入口，初始化 + 主循环
│   ├── display_task.cpp   # 显示任务 (LVGL)
│   ├── sensor_task.cpp    # 传感器采集任务
│   ├── audio_task.cpp     # 音频采集与播放
│   ├── ai_client.cpp      # WiFi + HTTP/WS 云端AI通信
│   ├── ui/
│   │   ├── watch_face.cpp # 表盘界面

├── platformio.ini

```

## 八、实物组装步骤

1. **PCB/洞洞板布局**：按照连接表规划元器件位置，注意模拟/数字区域分离
2. **焊接电源部分**：TP4056 → 电池插座 → 3.3V 稳压 → 去耦电容
3. **焊接 I2C 总线**：BME280 + DS3231 + 上拉电阻
4. **焊接 SPI 显示屏**：注意 FPC 排线方向
5. **焊接音频部分**：麦克风远离扬声器，避免啸叫
6. **焊接按键**：按键安装在手表侧面
7. **上电测试**：先不焊接电池，USB 供电测试所有功能
8. **组装外壳**：3D 打印外壳，安装表带
